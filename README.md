# TasVisAn

TasVisAn is a Python library for data reduction of the triple-axis spectrometers Taipan and Sika at ACNS, ANSTO.

## Installation

Use the package manager [pip](https://) to install TasVisAn.

```bash
pip install TasVisAn
```

## Usage

```python
import TasVisAn


```
